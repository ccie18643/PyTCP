Before running any of the examples please make sure to:
 - Go to the stack root directory (it is called 'PyTCP').
 - Run 'sudo make tap' command to create the tap7 interface and assign it to 'br0' bridge.
 - Run 'make' command to create the proper virtual environment.
 - Run '. venv/bin/activate' command to start stack virtual environment.
 - Execute any example, eg. 'example/run_stack.py'.
 - Hit Ctrl-C to stop it.
